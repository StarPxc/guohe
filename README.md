# 果核app后台
**提供果核app的后台数据接口，包括，课表信息，成绩信息，绩点计算等等**
### 果核app介绍
果核是一款基于Python+SSM+Android+Ngnix的高校生活平台，目前致力于提供江苏科技大学在校大学生的课表，成绩，绩点查询服务，并集成了一些大学生感兴趣的内容
